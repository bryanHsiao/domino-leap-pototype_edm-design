# DM 產生器網頁應用 - 實施計劃

## Context

使用者需要一個網頁工具，讓使用者上傳圖片與文字後，即時預覽並匯出專業的宣傳 DM（課程宣傳、簡章、活動公告等）。需整合 `nextlevelbuilder/ui-ux-pro-max-skill` 的設計系統原則。

## 技術方案

**單一 HTML 檔案**，內嵌 CSS + JS，零框架依賴。使用 html2canvas CDN 實現圖片匯出。

## 設計方向（依據 UI UX Pro Max Skill）

- **UI 風格**: Glassmorphism + Flat Design（SaaS 推薦模式）
- **色彩方案**: 深色工作區（設計工具風格）+ 明亮預覽區
  - Primary: `#2563EB`（信任藍）
  - Accent: `#EA580C`（行動導向橘）
  - 工作區背景: `#0F172A`（深藍灰）
  - 預覽背景: `#F8FAFC`
- **字體配對**: Noto Sans TC（中文介面）+ Poppins（UI 標題）+ Open Sans（正文）
- **UX 指南遵循**:
  - 觸摸目標 ≥ 44×44px
  - 過渡動畫 200-300ms
  - `prefers-reduced-motion` 支援
  - WCAG AA 對比度 4.5:1
  - 響應式斷點: 375px / 768px / 1024px / 1440px

## 功能規格

### 1. 版面配置（左右分割）
- **左側面板**: 編輯控制區（模板選擇、文字輸入、圖片上傳、尺寸選擇）
- **右側面板**: 即時預覽區（隨輸入內容即時更新）

### 2. 尺寸選項
- A4 直式 (210×297mm → 預覽按比例縮放)
- 社群正方形 (1080×1080px)
- 社群橫幅 (1200×628px)
- 16:9 寬幅 (1920×1080px)

### 3. 六種 DM 模板

| # | 名稱 | 風格 | 適用場景 |
|---|------|------|----------|
| 1 | 現代簡約 | 大面積留白、粗體標題、幾何色塊 | 課程宣傳 |
| 2 | 漸層活力 | 對角線元素、漸層覆蓋、動感排版 | 活動公告 |
| 3 | 經典專業 | 結構化版面、企業配色、分欄佈局 | 正式簡章 |
| 4 | 雜誌風格 | 大圖背景、文字疊加、編輯排版 | 品牌宣傳 |
| 5 | 繽紛卡片 | 圓角卡片、多彩配色、友善風格 | 社群貼文 |
| 6 | 暗色質感 | 深色底、金屬漸層、高端質感 | 高端課程/VIP |

### 4. 使用者可編輯項目
- 主標題、副標題、內文描述、CTA 按鈕文字
- 上傳圖片（拖放 + 點擊上傳）
- 主色調選擇器（覆蓋模板預設色）
- 背景色選擇

### 5. 匯出功能
- 使用 html2canvas（CDN 載入）將預覽區域渲染為 PNG
- 下載按鈕觸發檔案下載

## 檔案結構

```
index.html  ← 唯一檔案（內嵌所有 CSS + JS）
PLAN.md     ← 本計劃檔
```

## 實作步驟

1. 建立 HTML 骨架（左右面板佈局）
2. 實作 CSS 設計系統（變數、Glassmorphism、深色工作區）
3. 實作六個模板的 CSS 樣式
4. 實作 JS 邏輯：
   - 模板切換
   - 圖片上傳（FileReader + drag/drop）
   - 即時預覽更新（input 事件監聽）
   - 尺寸切換
   - 顏色自訂
   - html2canvas 匯出下載
5. 響應式適配（行動裝置改為上下佈局）

## 驗證方式

1. 用 Preview MCP 啟動本地伺服器查看頁面
2. 測試每個模板的切換與預覽
3. 測試圖片上傳（拖放 + 點擊）
4. 測試文字輸入即時更新
5. 測試匯出下載 PNG
6. 測試響應式（桌面 + 行動裝置視窗）
